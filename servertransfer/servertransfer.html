ServerTransfer

戻る

アドオンの概要

ServerTransferは、UIから簡単に別サーバーへ移動できるアドオンです。Bedrock Dedicated Server (BDS) などで複数のワールドを繋ぐハブサーバーを構築する際に役立ちます。

アドオンの入手

ServerTransferは以下のボタンからダウンロードできます。

【mcaddon】ServerTransfer_v1.2.0 【zip】ServerTransfer_v1.2.0

・v1.2.0: UUIDの修正(別アドオンと干渉する不具合を修正しました)

・v1.1.0: セレクターの不具合修正&アップデート時にアドオンの故障のリスクを軽減

機能と設定

このアドオンは、簡単なコマンドでサーバー移動UIを表示し、設定ファイル `config.js` を編集することで移動先サーバーを自由にカスタマイズできます。

設定方法

アドオン内の `scripts/config.js` ファイルを編集して、移動先のサーバーリストを設定します。`name` にはUIに表示する名前、`ip` と `port` には接続先サーバーの情報を入力してください。

// scripts/config.js の例
export const config = [
    {
        name: 'ロビーサーバー',
        ip: '000.000.000.000',
        port: 19132,
    },
    {
        name: 'ゲームサーバー',
        ip: '111.111.111.111',
        port: 19133,
    },
];

使用方法

以下のコマンドを実行することで、対象のプレイヤーにサーバー移動UIを表示できます。

/scriptevent servertransfer:ui <ターゲットセレクター>

実行例:
・自分にUIを表示: `/scriptevent servertransfer:ui @s`
・全てのプレイヤーに表示: `/scriptevent servertransfer:ui @a`

導入方法

以下の手順でアドオンを導入してください：

  • ワールド設定でチートを有効にする
  • 「ベータAPI」を有効にする
注意: このアドオンはプレイヤーを別サーバーに転送する機能を使用します。そのため、Bedrock Dedicated Server (BDS) など、サーバー転送が許可されている環境でのみ動作します。通常のローカルワールドでは機能しません。
